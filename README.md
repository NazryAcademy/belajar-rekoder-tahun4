# Belajar Rekoder Pro v2.0

## Sebelum upload ke GitHub
1. Buka `js/config.js`.
2. Gantikan `PASTE_PUBLISHABLE_KEY_DI_SINI` dengan Publishable Key Supabase yang bermula `sb_publishable_`.
3. Jangan gunakan Secret Key atau service_role.
4. Tukar kata laluan admin jika perlu.
5. Upload semua fail dan folder ke repository GitHub.

Netlify akan deploy secara automatik.
